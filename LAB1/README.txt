In order to compile and run successfully the software, c++11 should be used. Thus, the flag -std=c++11 should be added to the compile command as follows:

$ g++ -std=c++11 main.cpp LAB1.cpp network.cpp

Should the compiling succeed, default namefile "./a.out" will be used to create the executable.
So as to personalize it, do as follows, removing the quotes:

$ g++ -std=c++11 -o "YOUR_FILE_NAME" main.cpp LAB1.cpp network.cpp
